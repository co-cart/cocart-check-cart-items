<p align="center">This is a feature plugin that checks items in the cart for validity and stock. Updates the cart before the cart returns what remains and returns a notice for each item changed.</p>

## Requirement

You will need to be using [CoCart **v2.1**](https://wordpress.org/plugins/cart-rest-api-for-woocommerce/) and up before installing this add-on.

## Download

[Click here to download](https://github.com/co-cart/cocart-check-cart-items/archive/master.zip) the latest release package of CoCart.
